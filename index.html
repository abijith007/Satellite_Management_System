The Satellite Management System is a desktop ready web application that facilitates optimisation of the process of managing of satellite information on three fronts. From a End user's perspective, the application allows them to track and view the satellite's position, orbit, field of view and time of passing of satellite above a specific location on Earth, and view various orbit parameters, right from their user account. From the satellite manufacturer's perspective, it allows organization to browse through the satellite manufactured by them and also track their orbit in order to detect some defects in the satellite's orbit (if any). The owner organization can add, modify and update the satellite's information which they own and can view satellite's owned by other organizations. The admin of the SATMAN system has all the functionality that a satellite owner has, as well as an admin can add/remove users and organizations from the database. Other supporting functionality includes automatically procuring the Two Line Element (TLE) data from NORAD's satellite database, in order to update the satellite's orbit plotting, from the internet. The current systems available are either for only end users or only for organizations. The SATMAN system, aims to unify this process into one smooth application which can be used by all classes of access holders with the required level of authorization.
